# راهنمای راه‌اندازی V2Ray VPN

این پروژه اکنون از V2Ray برای ایجاد تونل VPN استفاده می‌کند. V2Ray یک پلتفرم پیشرفته برای ساخت پروکسی است که از پروتکل‌های مختلفی مانند VMess پشتیبانی می‌کند.

## پیش‌نیازها

- سرور لینوکس با IP استاتیک و اینترنت بین‌المللی
- پورت 443 باید باز باشد (یا پورت دیگری که در کانفیگ تنظیم کنید)

## مرحله ۱: نصب V2Ray

1. اسکریپت `install-v2ray.sh` را اجرا کنید:
   ```bash
   bash install-v2ray.sh
   ```

## مرحله ۲: کپی کانفیگ

1. فایل `config/v2ray-config.json` را به `/usr/local/etc/v2ray/config.json` کپی کنید.
2. سرویس V2Ray را راه‌اندازی کنید:
   ```bash
   systemctl enable v2ray
   systemctl start v2ray
   ```

## مرحله ۳: مدیریت کاربران

1. پنل مدیریت را اجرا کنید:
   ```bash
   cd admin-panel
   pip install -r requirements.txt
   export VPN_ADMIN_USER=admin
   export VPN_ADMIN_PASS=yourpassword
   python app.py
   ```
   پنل روی http://localhost:5000 اجرا می‌شود.

2. در پنل، کاربران جدید اضافه کنید. برای هر کاربر، UUID تولید می‌شود و لینک VMess ایجاد می‌گردد.

## مرحله ۴: توزیع به کاربران

- فایل کانفیگ کلاینت (vmess://...) را دانلود کرده و به کاربران بدهید.
- کاربران می‌توانند از اپ V2Ray یا کلاینت‌های سازگار استفاده کنند.

## مرحله ۵: محدودیت بر اساس حجم مصرفی

برای محدود کردن هر حساب به یک سقف مصرف (مثلاً ۵ گیگ)، از اسکریپت `scripts/limit-traffic.py` استفاده کنید.

1. در پنل هنگام ساخت کاربر، مقدار "محدودیت (گیگ)" را وارد کنید.
2. اسکریپت با خواندن آمار V2Ray و مقایسه با مقدار وارد شده، اگر مصرف کاربر از سقف بیشتر شود آن کاربر را غیرفعال می‌کند.

برای اجرای این اسکریپت بطور مداوم (مثلاً هر ۵ دقیقه)، می‌توانید از cron استفاده کنید:

```cron
*/5 * * * * python /workspaces/vpn/scripts/limit-traffic.py >> /var/log/v2ray/traffic-limit.log 2>&1
```

> دقت کنید که `v2ray-config.json` باید بخش `api` و `stats` را داشته باشد (این بخش هم‌اکنون در فایل تنظیم شده است).

## نکات امنیتی

- UUIDها منحصر به فرد هستند و برای احراز هویت استفاده می‌شوند.
- تاریخ انقضا برای کنترل دسترسی استفاده می‌شود.
- پورت 443 را در فایروال باز کنید.

برای جزئیات بیشتر، به مستندات V2Ray مراجعه کنید.
