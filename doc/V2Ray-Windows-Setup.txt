راهنمای نصب و راه‌اندازی V2Ray روی ویندوز

1. دانلود V2Ray:
- به سایت رسمی V2Ray (https://www.v2ray.com/) یا گیت‌هاب (https://github.com/v2ray/v2ray-core/releases) بروید.
- نسخه ویندوز را دانلود کنید (معمولاً فایل zip).

2. استخراج فایل:
- فایل zip را در یک پوشه دلخواه (مثلاً C:\V2Ray) استخراج کنید.

3. تنظیم فایل کانفیگ:
- فایل config.json را با تنظیمات سرور خود جایگزین کنید (یا ویرایش کنید).
- فایل config را معمولاً از مدیر سرور دریافت می‌کنید.

4. اجرای V2Ray:
- وارد پوشه V2Ray شوید.
- فایل v2ray.exe را اجرا کنید:
  v2ray.exe -config config.json

5. تنظیم فایروال:
- اگر فایروال ویندوز فعال است، اجازه دسترسی به v2ray.exe را بدهید.

6. استفاده از کلاینت (مثلاً V2RayN):
- نرم‌افزار V2RayN را دانلود و نصب کنید.
- کانفیگ سرور را وارد کنید (import config).
- اتصال را برقرار کنید.

7. تست اتصال:
- مرورگر را باز کنید و سایت‌های فیلتر شده را تست کنید.

نکات:
- اگر TLS فعال است، فایل certificate را در مسیر مناسب قرار دهید.
- اگر از پروتکل‌های خاص (ws، tcp، kcp) استفاده می‌کنید، تنظیمات مربوطه را در config.json وارد کنید.
- برای اجرای خودکار، می‌توانید v2ray.exe را به Task Scheduler اضافه کنید.

مشکلات رایج:
- اگر خطای TLS handshake timeout دریافت کردید:
  - پورت و دامنه را بررسی کنید.
  - گواهی SSL را چک کنید.
  - اینترنت و فایروال را بررسی کنید.

پایان راهنما.





.\v2ray.exe run -config .\config.json
